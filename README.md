# 📘 Blogfolio-Rails

Un proyecto de **Blogfolio** simple desarrollado con **Ruby on Rails**, integrando Docker para un entorno de desarrollo portable. Ideal como portafolio personal o blog técnico.

---

## 🧰 Tecnologías utilizadas

- 💎 **Ruby on Rails**
- 🐳 **Docker** + **Kamal**
- 🐘 **PostgreSQL**
- ⚙️ **StimulusJS**
- 📦 **Yarn / Node.js**
